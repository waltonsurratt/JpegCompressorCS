# Jpeg Compressor CS
This a new project for a native, Windows-based application written in C# that compresses JPEG images and offers flexibility with adjusting the quality level as desired and reducing overall file size. This application is an recreation/expansion of the C++ utility developed (JpegCompressor) - this utility offers nearly the same exact benefits in JPEG compression options and optimizes memory usage, but comes with a more modern-looking UI for Windows 10/11 systems. This application is also built on .NET 10, bringing it up-to-date with all current Microsoft standards.

# Version: 1.2.0
The tool currently includes the following features (initial release):
* Batch file processing
* Error handling for non-JPEG files
* File overwrite behavior
* Image Quality slider bar (linked to compression rate)
* Modern UI Design
